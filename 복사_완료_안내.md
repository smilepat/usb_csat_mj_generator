## F 드라이브에서 사용하는 방법

### 1단계: Node 모듈 설치
```bash
cd F:\csat_mj_generator\web-app
npm install
cd client
npm install
```

### 2단계: 시스템 시작
```bash
F:\csat_mj_generator\start.bat
```

또는 루트 디렉토리에서 `start.bat` 파일을 더블클릭하세요.

### 3단계: 브라우저 접속
- **프론트엔드:** http://localhost:3000
- **백엔드 API:** http://localhost:3001

## 주의사항

⚠️ **처음 실행 시 반드시 `npm install`을 실행하세요!**

node_modules 폴더가 복사되지 않았으므로, 의존성 패키지를 재설치해야 합니다.

```bash
# 백엔드 의존성 설치
cd F:\csat_mj_generator\web-app
npm install

# 프론트엔드 의존성 설치
cd F:\csat_mj_generator\web-app\client
npm install
```

설치 완료 후 `start.bat`을 실행하면 정상 작동합니다.

## 문제 해결

### 포트 충돌 시
```bash
stop.bat
start.bat
```

### 수동으로 프로세스 종료
```bash
# 포트 3001 (백엔드) 종료
netstat -ano | findstr :3001
taskkill /F /PID [PID번호]

# 포트 3000 (프론트엔드) 종료
netstat -ano | findstr :3000
taskkill /F /PID [PID번호]
```
